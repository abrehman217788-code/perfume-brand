⭐ Method 1: Browser Print to PDF (EASIEST)

Time Required: 30 seconds | Difficulty: Very Easy

For Windows/Linux:

  1. Open the website you want to save (any of your portfolio pages)
  2. Press Ctrl + P on your keyboard
  3. Look for the printer dropdown (usually shows "Print" or your printer name)
  4. Change it to "Save as PDF"
  5. Click the filename field and enter: AB_Rehman_Portfolio
  6. Click "Save" button

For Mac:

  1. Open the website you want to save
  2. Press Cmd + P on your keyboard
  3. In the popup, click the "PDF" dropdown at bottom left
  4. Select "Save as PDF"
  5. Enter filename and click "Save"

🌐 Method 2: Online PDF Converter (No Installation)

Time Required: 1 minute | Difficulty: Easy

  1. Go to any online HTML to PDF converter:
    • PDF24 Tools
    • HTML to PDF
    • Convertio
  2. Upload your HTML file or paste the file path
  3. Click "Convert" button
  4. Download the generated PDF
✓ Advantages: No software installation needed, free, works on any device

⚙️ Method 3: Command Line (Advanced)

Time Required: 5 minutes | Difficulty: Advanced

Using wkhtmltopdf (Free):

  1. Download from: wkhtmltopdf.org
  2. Install it on your computer
  3. Open Command Prompt (Windows) or Terminal (Mac/Linux)
  4. Run this command: 
    wkhtmltopdf C:\path\to\portfolio-website\index.html output.pdf
  5. Your PDF will be created in the same folder

❓ Frequently Asked Questions

Q: What file size should I expect?

Most portfolio PDFs are 200-400 KB each. All 4 projects combined will be under 2 MB.

Q: Why is my PDF missing content?

Make sure "Background Graphics" is enabled in the print dialog. Some printers disable this by default.

Q: Can I edit the PDF after saving?

Yes, use free tools like PDFtk or SmallPDF.

Q: Which method is fastest?

Browser Print to PDF is fastest for a single page (30 seconds). Automated tools are better if you need to convert many files.
